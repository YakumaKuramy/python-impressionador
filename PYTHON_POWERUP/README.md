# Módulo: Python PowerUp

Este módulo focou na automação de tarefas utilizando bibliotecas externas.

## 🛠️ Tecnologias

- Python 3.13
- Pandas

## 🧠 O que aprendi?

- Como ler arquivos `.csv` de forma dinâmica com `pathlib`.
- Como evitar erros comuns de diretório no Windows e Linux.
- Boas práticas de estruturação de scripts.

## 🚀 Como executar

1. Navegue até a pasta: `cd PYTHON-POWERUP`
2. Execute o script: `python automation.py`
